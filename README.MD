Contexto Profissional

O Café Central é uma cafeteria fictícia localizada em uma região de grande circulação de pessoas. O negócio oferece cafés especiais, bebidas quentes e frias, lanches e sobremesas, atendendo tanto para consumo local quanto para retirada.
Atualmente, o Café Central não possui um site institucional. O proprietário deseja fortalecer a identidade da marca, divulgar seus produtos e facilitar o contato com os clientes por meio de um site institucional autoral.

Objetivo da Situação de Aprendizagem - Café Central

O objetivo desta Situação de Aprendizagem é possibilitar que o aluno analise de forma autônoma o contexto do cliente, levante
requisitos, defina o escopo e planeje um site institucional, aplicando os mesmos procedimentos observados no projeto guiado.

Etapa 1 - Definição do Escopo do Site

Objetivo do site: disponibilizar uma visão geral sobre a cafeteria.

Público-alvo: público em geral.

Páginas da cafeteria definidas:

Home
Sobre
Cardápio
Contato

Etapa 2 - Levantamento e Definição de Requisitos

Requisitos Funcionais

Exibir informações relacionadas a Cafeteria
Listar Produtos Disponíveis no site
Disponibilizar formulário de contato
Garantir navegação funcional entre páginas

 Requisitos Não Funcionais

Layout organizado,rápido e intuitivo
Código HTML semântico, legível e indentado
Compatibilidade com navegadores modernos

Requisitos Estruturais
Menu de Navegação presente em todas as páginas
Estrutura de pastas padronizada


Etapa 3 - Organização do Ambiente de Desenvolvimento e Estrutura Técnica do Projeto

Organização do ambiente de desenvolvimento:
IDE utilizada: Visual Studio Code
Pasta raiz do projeto criada: cafecentral-site
Padrão de nomenclatura aplicado (letras minúsculas, sem espaços)
Projeto preparado para separação entre HTML, CSS e JavaScript

Estrutura técnica definida e organizada (diretórios criados):
/css com ‘x’ arquivos de estilo (layout e identidade visual)
/js com ‘x’ scripts JavaScript (interações futuras)
/img com ‘x’ imagens institucionais
/pages com ‘x’ páginas internas do site

Arquivos definidos e criados:
index.html - página inicial (Home)
/pages/sobre.html
/pages/cardapio.html
/pages/contato.html

Estrutura de navegação planejada:
Menu principal presente em todas as páginas
Links funcionais entre páginas institucionais
